# Requirement
Vi : Hệ điều hành Windows 10 x64 hoặc x86  
En : Microsoft Windows 10 Operating System x64 or x86

# Installation
Vi :

En:  
Download and extract the file "Flap.rar". Run the file "FlappyBird" to play the game
